┌──────────────────────────────────────────────────────────────────────────┐
│ 📋 PREREQUISITES                                                         │
└──────────────────────────────────────────────────────────────────────────┘

   ⚠️  ONLY NEEDED FOR REMOTE API!
   Your current GitHub workflow does NOT require Node.js!

   Why Node.js? Remote API tools are Node.js apps that run a WebSocket server
   to sync files. Without it, Remote API won't work (but GitHub method is fine).

   Required software (install these first if you don't have them):
   • Node.js (includes npm): https://nodejs.org/
   • Git: https://git-scm.com/downloads
   
   Verify you have them:
   ┌────────────────────────────────────────────────────────────────────┐
   │ node --version
   │ npm --version
   │ git --version
   └────────────────────────────────────────────────────────────────────┘

   Note: npm, git, cd are cross-platform tools (work in PowerShell, bash, etc.)

┌──────────────────────────────────────────────────────────────────────────┐
│ 📋 INSTALLATION (5 MINUTES)                                              │
└──────────────────────────────────────────────────────────────────────────┘

   1. INSTALL TYPESCRIPT TEMPLATE
      PowerShell:
      ┌────────────────────────────────────────────────────────────────────┐
      │ git clone https://github.com/bitburner-official/typescript-template bitburner-dev
      │ cd bitburner-dev
      │ npm install
      └────────────────────────────────────────────────────────────────────┘

   2. START THE SYNC TOOL
      ┌────────────────────────────────────────────────────────────────────┐
      │ npm run watch:remote
      └────────────────────────────────────────────────────────────────────┘
      
      Expected output:
      ✅ Server is ready, running on 12525!

   3. CONNECT BITBURNER
      In-game steps:
      • Options → Remote API
      • Hostname: localhost
      • Port: 12525
      • Click "Connect"
      
      Expected result:
      ✅ Connected to Remote API

   4. TEST IT
      • Edit any file in bitburner-dev/src/
      • Save (Ctrl+S)
      • Check Bitburner - file appears instantly!

┌──────────────────────────────────────────────────────────────────────────┐
│ 🚨 TROUBLESHOOTING                                                       │
└──────────────────────────────────────────────────────────────────────────┘

   ❌ "Failed to connect"
      → Verify npm run watch:remote is running
      → Check port 12525 is shown in console
      → Use "localhost" not "127.0.0.1"

   ❌ "Scripts not syncing"
      → Check Remote API shows "Connected" in-game
      → Verify files are in src/ directory
      → Look for errors in npm run watch:remote console

   ❌ "Connection lost after sleep"
      → In Bitburner: Options → Remote API → Connect (again)
      → No need to restart npm run watch:remote
