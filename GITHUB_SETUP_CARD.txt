╔══════════════════════════════════════════════════════════════════════════╗
║                 BITBURNER GITHUB DEPLOYMENT CARD                         ║
║                         Quick Reference                                  ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  📦 STEP 1: PREPARE SCRIPTS                                              ║
║  ───────────────────────────                                             ║
║  > .\Prepare-GitHubUpload.ps1                                            ║
║                                                                          ║
║  Result: bitburner-github/ folder created with all scripts              ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  🌐 STEP 2: CREATE GITHUB REPO                                           ║
║  ──────────────────────────                                              ║
║  1. Go to: https://github.com/new                                        ║
║  2. Name: bitburner-scripts                                              ║
║  3. Visibility: ☑ Private (FREE!) or ☐ Public                            ║
║  4. Click "Create repository"                                            ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  ⬆️  STEP 3: UPLOAD FILES                                                 ║
║  ──────────────────────                                                  ║
║  1. Open your new repo on GitHub                                         ║
║  2. Drag & drop files from bitburner-github/ folder                      ║
║  3. Commit: "Initial upload"                                             ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  🔗 STEP 4: GET YOUR URL                                                 ║
║  ─────────────────────                                                   ║
║  1. Open any .js file in repo                                            ║
║  2. Click "Raw" button                                                   ║
║  3. Copy URL up to /main/                                                ║
║                                                                          ║
║  Format: https://raw.githubusercontent.com/USERNAME/REPO/main            ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  ⚙️  STEP 5: CONFIGURE UPDATE SCRIPT                                     ║
║  ───────────────────────────────                                         ║
║  1. Open bitburner-update.js in bitburner-github/                        ║
║  2. Line 14: Update baseUrl with your URL from Step 4                    ║
║  3. Save and upload to GitHub (overwrite existing)                       ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  🎮 STEP 6: FIRST RUN IN BITBURNER                                       ║
║  ──────────────────────────────                                          ║
║  In Bitburner terminal:                                                  ║
║                                                                          ║
║  > wget YOUR_URL/bitburner-update.js bitburner-update.js                 ║
║  > run bitburner-update.js --all                                         ║
║                                                                          ║
║  ✓ All scripts downloaded!                                               ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  🔄 DAILY WORKFLOW                                                       ║
║  ────────────────                                                        ║
║  1. Edit scripts locally                                                 ║
║  2. Push to GitHub                                                       ║
║  3. In game: run bitburner-update.js --all                               ║
║  4. Done! 🎉                                                             ║
║                                                                          ║

║  Scripts not updating?                                                   ║
║    → run global-kill.js                                                  ║
