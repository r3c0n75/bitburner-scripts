╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  📝 STEP 1: CREATE GITHUB REPO                                           ║
║  ──────────────────────────────                                          ║
║  1. Go to: https://github.com/new                                        ║
║  2. Name: bitburner-scripts                                              ║
║  3. Visibility: ☑ Private (FREE!)                                        ║
║  4. Initialize: Leave unchecked                                          ║
║  5. Create → Copy repo URL                                               ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  🚀 STEP 2: FIRST PUSH (PowerShell)                                      ║
║  ───────────────────────────────────                                     ║
║  PS> .\Push-ToGitHub.ps1 `                                               ║
║        -RepoUrl "https://github.com/USER/bitburner-scripts.git" `        ║
║        -FirstTime                                                        ║
║                                                                          ║
║  ✓ Creates .gitignore                                                    ║
║  ✓ Initializes Git                                                       ║
║  ✓ Pushes organized folders                                              ║
║  ✓ Shows your raw GitHub URL                                             ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  ⚙️  STEP 3: CONFIGURE UPDATE SCRIPT                                     ║
║  ────────────────────────────────                                        ║
║  1. Open: bitburner-update.js                                            ║
║  2. Line 15: Update baseUrl                                              ║
║     const baseUrl = "https://raw.githubusercontent.com/USER/REPO/main";  ║
║  3. Save                                                                 ║
║  4. Push:                                                                ║
║     PS> .\Push-ToGitHub.ps1 -RepoUrl "URL" -CommitMessage "Config"      ║
║                                                                          ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  🎮 STEP 4: FIRST RUN IN BITBURNER                                       ║
║  ──────────────────────────────────                                      ║
║  > wget YOUR_RAW_URL/bitburner-update.js bitburner-update.js             ║
║  > run bitburner-update.js --all                                         ║
║                                                                          ║
║  ✓ Downloads all scripts from organized folders                          ║
║  ✓ Saves flat to Bitburner home                                          ║
║                                                                          ║
